# State Diagram: Жизненный цикл трансформации Фионы

## Обзор

Эта диаграмма состояний показывает жизненный цикл Принцессы Фионы, которая меняет свою форму в зависимости от времени суток.

## Состояния

| Состояние | Описание | Цвет фона |
|-----------|-------------|------------------|
| Человек | Фиона в человеческой форме (являетсяОгром = false) | Default |
| Огр | Фиона в форме огра (являетсяОгром = true) | LightGreen |

## Переходы состояний

### Начальное состояние
- [*] --> Человек : Начало дня

### Переходы по времени суток
| Из состояния | В состояние | Условие | Действие |
|--------------|-------------|---------|----------|
| Человек | Огр | Заход солнца | трансформироваться() |
| Огр | Человек | Восход солнца | трансформироваться() |

### Конечное состояние
- Отсутствует (цикличный процесс)

## Вложенные состояния

### Состояние "Человек"
| Подсостояние | Описание | Атрибуты |
|--------------|-------------|----------|
| ДневноеСостояние | Активность в дневное время | являетсяОгром = false |

### Состояние "Огр"
| Подсостояние | Описание | Атрибуты |
|--------------|-------------|----------|
| НочноеСостояние | Активность в ночное время | являетсяОгром = true |

## Ключевые моменты

- **Цикличность:** Трансформации происходят ежедневно при смене времени суток
- **Автоматический переход:** Фиона не контролирует трансформацию — она происходит автоматически
- **Дневное состояние:** Человеческая форма, стандартное поведение принцессы
- **Ночное состояние:** Форма огра, проявление проклятия

## Сравнение с "Курочкой Рябой"

| Параметр | Курочка Ряба (Золотое яйцо) | Шрек (Фиона) |
|----------|------------------------------|--------------|
| **Тип процесса** | Линейный (целое → разбитое) | Цикличный (человек ↔ огр) |
| **Конечное состояние** | Разбитое → [*] | Нет конечного состояния |
| **Условие перехода** | сила >= 100 | Время суток (заход/восход) |
| **Действие при переходе** | воздействовать() | трансформироваться() |
| **Точка выбора** | Да (choice) | Нет |

## Диаграмма

![Диаграмма состояний Фионы](shrek_state_diagram.png)

```plantuml
@startuml
skinparam state {
  BackgroundColor<<Ogre>> #lightgreen
}

title Жизненный цикл трансформации Фионы

[*] --> Человек : Начало дня

state "Форма Человека" as Человек {
    [*] --> ДневноеСостояние
    state ДневноеСостояние {
        являетсяОгром = false
    }
}

state "Форма Огра" as Огр <<Ogre>> {
    [*] --> НочноеСостояние
    state НочноеСостояние {
        являетсяОгром = true
    }
}

' Переходы между состояниями
Человек --> Огр : Заход солнца\nтрансформироваться()
Огр --> Человек : Восход солнца\nтрансформироваться()

note right of Человек
  **Дневное состояние**
  Фиона выглядит как человек
  Ведёт себя как принцесса
end note

note left of Огр
  **Ночное состояние**
  Фиона превращается в огра
  Проявляет истинную сущность
end note

note bottom of Человек --> Огр
  **Автоматическая трансформация**
  Происходит каждый день
  при заходе солнца
end note

@enduml
```
